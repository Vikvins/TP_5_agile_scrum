# TP_5_agile_scrum


## Sprint Review

**Закрытые задачи:**
- Рефакторинг логики оценки надёжности паролей
- Улучшение дизайна и оформление интерфейса
- Реализация копирования пароля в буфер обмена

**Выполнено:**
- Логика оценки надёжности паролей переработана и работает корректно
- Интерфейс стал более аккуратным и удобным для пользователя
- Пользователь может копировать пароли в буфер обмена

**Не завершённые задачи:**
- Реализовать логику генерации паролей
- Добавить оценку надёжности пароля
- Создать пользовательский интерфейс генерации паролей
- Добавить проверку введённых пользователем паролей




## Sprint Retrospective

**Что прошло хорошо:**
- Быстро удалось закрыть часть задач спринта
- Добавление меток упростило отслеживание прогресса

**Что пошло не так:**
- Не успела завершить часть задач
- Требуется больше времени на реализацию и тестирование функционала

**Что можно улучшить:**
- Планировать более реалистичные сроки для задач
- Разбивать крупные задачи на более мелкие
- Ставить приоритеты для ускорения завершения спринта
